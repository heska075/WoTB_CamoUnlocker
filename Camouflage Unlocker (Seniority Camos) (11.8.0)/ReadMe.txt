<--- Installation Guide --->

Steam:
    - Copy and paste the Data folder to the World of Tanks Blitz folder in your Steam game directory. Replace all existing files. (Current)
    - (For micropatches/automatic updates, copy the 'packs' folder here: 'C:\Users\[USER]\AppData\Local\wotblitz')

WGC:
    - Copy and paste the Data folder to [DRIVE]:\World_of_Tanks_Blitz_WGC\. Replace all existing files.
